# **MatrixOne v26.3.0.7 Release Notes**

Release Date: February 22, 2025
MatrixOne Version: v26.3.0.7

MatrixOne 3.0.7 introduces improvements and bug fixes that enhance usability. Below are the major updates.

### Key Bug Fixes

- Fix pipeline client leak
- Fix execute create as select in one transaction and enforce source SELECT privilege
- Fix force CTAS follow-up insert to run on one CN
- Fix UNION with ORDER BY FIELD() function
- Fix cdc truncate bug
